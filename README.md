💼 Job Portal Web Application

A dynamic Job Portal Web Application built using Java (JSP & Servlets), MySQL, and JDBC, designed to connect job seekers with recruiters. The system allows users to create profiles, explore job opportunities, and apply for jobs, while recruiters can post and manage job listings efficiently.

🚀 Features

👤 Job Seeker
User Registration & Login
Profile Management (Education, Experience, Certificates)
Search & View Jobs
Apply for Jobs
Track Applied Jobs

🏢 Recruiter
Recruiter Registration & Login
Post New Jobs
Edit/Delete Job Listings
View Applicants

📄 Job Management
Add, Update, Delete Jobs
Categorized Job Listings
Dynamic Data Rendering

🔐 Authentication & Security
Session-based Login System
Role-based Access Control (User / Recruiter)

🛠️ Tech Stack
Frontend: HTML, CSS, JAVASCRIPT, BOOTSTRAP, JSP
Backend: Java Servlets
Database: MySQL
Connectivity: JDBC
Server: Apache Tomcat

🔄 System Workflow
User registers and logs in
Recruiter posts job openings
Job seekers browse available jobs
Users apply for jobs
Data is stored in MySQL database
Servlets handle requests and forward responses to JSP

⚙️ Installation & Setup

Clone the repository:
git clone https://github.com/your-username/job-portal.git
Import project into Eclipse / IntelliJ IDEA
Configure Apache Tomcat Server

Create MySQL database:
CREATE DATABASE jobportal;
Import database.sql file into MySQL

Update database credentials in:
DBConnection.java
Run the project on Tomcat:
http://localhost:8080/JobPortal
